# منصة تطوير مكتبات أكاديمية الفلاح 2026

هذه نسخة نهائية قبل رفع GitHub Pages.

## الروابط الرئيسية
1. نظام تقييم المكتبات
2. جدول الدوام والمهام اليومية والمتابعة
3. صفحة مشاركة الإدارة
4. صفحة المقارنة والتقارير النهائية
5. واجهة تقييم AI

## الحفظ في Supabase
- التقييمات تحفظ في جدول `library_evaluations`.
- الدوام والمتابعة تحفظ في جدول `daily_work_logs`.
- تقارير الإدارة تحفظ في جدول `management_reports` عند تفعيل الصفحة.
- تقييم AI يحفظ في جدول `ai_reviews` عند تفعيل الصفحة.
- ملفات المدارس تحفظ في جدول `school_files`.

## تخزين الملفات
لكل مدرسة صفحات مستقلة لرفع:
- صور المكتبة → bucket: `library-images`
- النماذج وقوائم العمل → bucket: `library-documents`
- شهادات الأخصائي → bucket: `specialist-certificates`

كل صفحة تسمح برفع 10 ملفات في المرة الواحدة، وتظهر الملفات بعد الرفع في نفس الصفحة.

## التفعيل
1. شغّل ملف `supabase_setup_complete.sql` داخل SQL Editor في Supabase.
2. افتح `supabase-config.js`.
3. ضع `Publishable key` فقط مكان `PUT_PUBLISHABLE_KEY_HERE`.
4. لا تستخدم Secret key نهائياً.
5. ارفع الملفات إلى GitHub Pages.

## ملاحظة مهمة
تبقى نسخة محلية احتياطية في المتصفح، لكن الحفظ الدائم يتم في Supabase عند وضع المفتاح الصحيح وتفعيل السياسات.
