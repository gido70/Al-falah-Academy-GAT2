# نظام مكتبات أكاديمية الفلاح

نسخة إصلاح كاملة مستقرة:
- لا تظهر الأكواد على الصفحات.
- جميع الصفحات متصلة بـ Supabase.
- التقييمات، الأسابيع، الملفات، التقارير، وتقييم AI تحفظ في القاعدة.
- صفحة المدارس ترفع الصور والوثائق والشهادات إلى Supabase Storage.

## الرفع
ارفع محتويات هذا المجلد كاملة إلى مستودع GitHub الحالي `alfalah-libraries` واستبدل الملفات القديمة.

## ملاحظة
إذا كانت الجداول والسياسات منفذة مسبقًا في Supabase فلا تحتاج تشغيل SQL مرة أخرى.
