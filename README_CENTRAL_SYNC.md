# منصة أكاديمية الفلاح — نسخة المزامنة المركزية النهائية

هذه النسخة تعالج المشكلة جذريًا:

- جدول الدوام والأسابيع يحفظ في Supabase في جدول `daily_work_logs` حسب `week_number`.
- عند فتح الصفحة من الهاتف أو الكمبيوتر، يتم جلب الأسبوع من Supabase.
- تقييم المكتبات يحفظ في `library_evaluations` ويقرأ من القاعدة.
- صور المكتبات والنماذج والشهادات ترفع إلى Storage وتسجل في `school_files`.
- `localStorage` موجود فقط كنسخة احتياطية عند فشل الاتصال، وليس المصدر الرئيسي.

قبل الرفع أو بعده شغّل:
`supabase_central_sync_final.sql`
داخل Supabase SQL Editor.

ثم ارفع هذه الملفات كاملة إلى GitHub.
